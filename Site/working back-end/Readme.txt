To launch back-end:

1.Go to stuff directory -> cmd line -> type:
npm install http-server -g
(Downloads testnode server for directory)

2. stuff directory -> cmd line type http-server . -o
(This launches the test server for the recipes.html file)

3. Working back-end directory -> cmd line -> node app2.js
(Launches node back-end test server)

4. Click on Recipes.html in stuff directory
